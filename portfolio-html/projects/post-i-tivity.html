A progressive web app to upload images and posts that users can make as they explore Exeter University's campus and promote sustainability through improving mental health, exercise, community engagement, social welfare and awareness of the environment.

Project Overview

Post-i-tivity is a progressive web application designed to enhance the university experience at Exeter by encouraging students to explore the campus while promoting sustainability through various aspects of wellbeing. The app allows users to create and collect virtual postcards as they walk around campus, fostering a sense of community and environmental awareness.

The project was developed as part of the ECM2434 Software Engineering Group Project module at the University of Exeter. Our team, Exception Handler, worked collaboratively to design, develop, and deploy this application using modern web technologies and following agile development practices.

Key Features

Virtual Postcards

Create and collect virtual postcards by exploring different locations around the university campus.

Location-Based Content

Discover content specific to your location on campus, encouraging physical activity and exploration.

Sustainability Focus

Promotes environmental awareness and sustainable practices through engaging content and challenges.

Mental Health Support

Encourages activities that improve mental wellbeing through exercise, social interaction, and mindfulness.

Community Engagement

Connect with other students and share experiences through the platform, fostering a sense of community.

Progressive Web App

Works across all devices with a responsive design and can be installed on home screens for quick access.

Development Process

Our team followed an agile development methodology, using Trello for task management and regular sprint meetings to track progress. We began with a comprehensive problem analysis and requirements gathering phase, creating a MoSCoW matrix to prioritize features.

The design phase involved creating detailed UI/UX mockups in Figma, ensuring a user-friendly and visually appealing interface. We also developed an ER diagram to plan the database structure efficiently.

Development was split between frontend and backend teams, with regular integration to ensure smooth functionality. We implemented continuous integration and deployment practices to maintain code quality and facilitate quick updates.

Technologies Used

The project leverages a modern tech stack to deliver a seamless user experience:

React JavaScript HTML/CSS Figma Geolocation API Django Python RESTful API PostgreSQL Docker

Challenges & Solutions

Throughout the development process, we encountered several challenges:

  • Location Accuracy: Ensuring precise geolocation was challenging. We implemented a custom algorithm to improve accuracy and provide a better user experience.
  • Cross-Platform Compatibility: Making the app work seamlessly across different devices required extensive testing and responsive design implementation.
  • User Engagement: To keep users engaged, we incorporated gamification elements and regular content updates to encourage continued exploration.
  • Performance Optimization: We optimized image loading and database queries to ensure fast performance even with limited connectivity.
